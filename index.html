STEP 1 Monetate → Banner Clicks CSV
  1. Open Monetate → Analytics → Events
  2. Filter event: homepage_banner_click
  3. Set date range to your desired period
  4. Click Export → Analysis events table
  5. Save the file and upload it below ↓
💡 File will be named like Analysis_events_table_*.csv
STEP 2 Shopify → Sessions CSV
  1. Open Shopify Admin → Analytics → Reports
  2. Find report: Sessions by landing page
  3. Set the same date range as Monetate
  4. Click Export → Export CSV
  5. Save the file and upload it below ↓
💡 File will be named like Sessions_by_landing_page_*.csv
